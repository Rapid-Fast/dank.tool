## 📝 Notes

- **Windows Defender Warning**: The executable is not signed, so it may trigger warnings as an unknown source (false positive).
- **Running Python Scripts**: To run scripts directly, some modifications (e.g., paths) may be needed.
- **Console Tips**: Avoid spamming [ENTER] in the terminal to prevent incorrect input handling. If issues occur, restart the executable.
- **Pause & Resume**: If the console window is clicked, a white rectangle appears, pausing the task. Press the down arrow to continue.
- **Keyboard Interruptions**: Avoid [Ctrl + C] when the tool is active as it will close the program.
- **UI Refresh**: Resize the window and input `refresh` to realign the UI.
- **Startup Options**: Enable `force-startup-audio` in settings for animation and music on every run.
- **Developer Settings**: Options like `offline-src`, `offline-mode`, or `dev-branch` are for developers only.
- **Developer Documentation**: Available [here](https://github.com/SirDank/dank.tool#-danktool-documentation--for-devs--).

## 🔧 Features

- **Customization**: Server name, [Purpur](https://github.com/PurpurMC/Purpur) version, server RAM, cracked player support.
- **Multithreaded Downloading**: 2 threads for file downloading.
- **Automated Updates**: Uses [AutoPlug](https://github.com/Osiris-Team/AutoPlug-Client) for updating JDK, plugins, server jar, and more!
- **Hosting Solution**: [playit.gg](https://github.com/playit-cloud/playit-agent) integration.
- **Plugin Downloads**: Fetches plugins from [Spigot](https://github.com/search?q=repo%3ASirDank%2Fdank.tool+%22spigot_plugins+%3D+%7B%22&type=code) / [GitHub](https://github.com/search?q=repo%3ASirDank%2Fdank.tool+%22github+plugins%22&type=code).
- **Anti-Malware**: Includes [Minecraft anti-malware](https://www.spigotmc.org/resources/spigot-anti-malware.64982/).
- **Exploit Patches**: Automatically applies [Log4j](https://help.minecraft.net/hc/en-us/articles/4416199399693-Security-Vulnerability-in-Minecraft-Java-Edition) exploit patches.
- **Optimization Scripts**: Generates scripts with [Aikar's Flags](https://aikar.co/2018/07/02/tuning-the-jvm-g1gc-garbage-collector-flags-for-minecraft/) and [Etil's Flags](https://github.com/etil2jz/etil-minecraft-flags).
- **Java and Malware**: Quick installation scripts for Java and malware scanning.
- **AutoPlug Configuration**: Automatically configures AutoPlug for downloaded plugins.
- **Optimization**: Applies optimizations from [here](https://github.com/YouHaveTrouble/minecraft-optimization).
- **Chatroom Integration**: Integrated with dank.tool's chatroom via [dank.chatroom-plugin](https://github.com/SirDank/dank.chatroom-plugin).

## 🔧 Features

- **Scanning**: Generates and scans IPs on default Minecraft server ports (25565 for Java, 19132 for Bedrock).
